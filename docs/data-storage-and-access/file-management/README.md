# File Management

## Overview

File Management is a submodule of **Data Storage & Access** responsible for the complete lifecycle of physical database files on disk. It serves as the foundational layer between the DBMS engine and the operating system's file system — abstracting raw file I/O into structured, manageable units that the rest of the system can reason about.

Every database operation — from creating a new table to recovering from a crash — ultimately depends on File Management to correctly locate, open, read, write, and close the underlying files.
